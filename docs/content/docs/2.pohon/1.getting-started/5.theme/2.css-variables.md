```css [app/assets/css/main.css]
@import "tailwindcss";
```

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --ui-primary: black;

.dark {
  --ui-primary: white;
}
```

## Radius

Nuxt UI overrides Tailwind CSS's default `rounded-*` utilities with a unified border radius system, allowing you to use regular [border radius utilities](https://tailwindcss.com/docs/border-radius) like `rounded-xs` or `rounded-2xl`:

::code-preview
[xs]{class="border-2 border-border-accented text-sm px-4 py-1.5 inline-block rounded-xs mr-2"}
[sm]{class="border-2 border-border-accented text-sm px-4 py-1.5 inline-block rounded-sm mr-2"}
[md]{class="border-2 border-border-accented text-sm px-4 py-1.5 inline-block rounded-md mr-2"}
[lg]{class="border-2 border-border-accented text-sm px-4 py-1.5 inline-block rounded-lg mr-2"}
[xl]{class="border-2 border-border-accented text-sm px-4 py-1.5 inline-block rounded-xl mr-2"}
[2xl]{class="border-2 border-border-accented text-sm px-4 py-1.5 inline-block rounded-2xl mr-2"}
[3xl]{class="border-2 border-border-accented text-sm px-4 py-1.5 inline-block rounded-3xl mr-2"}

#code
```vue
<template>
  <div class="rounded-xs">xs</div>
  <div class="rounded-sm">sm</div>
  <div class="rounded-md">md</div>
  <div class="rounded-lg">lg</div>
  <div class="rounded-xl">xl</div>
  <div class="rounded-2xl">2xl</div>
  <div class="rounded-3xl">3xl</div>
</template>
```
::

These utility classes are calculated based on a global `--ui-radius` CSS variable, which defines the base radius value applied across all components for a consistent look.

```css
:root {
  --ui-radius: 0.25rem;
}
```

::tip
You can customize the base radius value in your `main.css` file:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --ui-radius: 0.5rem;
}
```
::

::note
Try the :prose-icon{name="i-lucide:swatch-book" class="text-primary"} theme picker in the header above to change the base radius value.
::

## Container

Nuxt UI provides a `--ui-container` CSS variable that controls the maximum width of the [Container](/docs/pohon/components/container) component.

```css
:root {
  --ui-container: var(--container-7xl);
}
```

::tip
You can customize this value in your `main.css` file to adjust container widths consistently throughout your application:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme {
  --container-8xl: 90rem;
}

:root {
  --ui-container: var(--container-8xl);
}
```
::

## Header

Nuxt UI provides a `--pohon-header-height` CSS variable that controls the height of the [Header](/docs/pohon/components/header) component.

```css
:root {
  --pohon-header-height: --spacing(16);
}
```

::tip
You can customize this value in your `main.css` to adjust header height consistently throughout your application:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --pohon-header-height: --spacing(24);
}
```
::
